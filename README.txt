===========================================
DERIV TRADING PLATFORM - SETUP INSTRUCTIONS
===========================================

1. EDIT config.js
   - Open config.js in a text editor
   - Replace "16929" with YOUR Deriv App ID
   - Save the file

2. GET YOUR APP ID
   - Go to https://developers.deriv.com
   - Log in with your Deriv account
   - Register a new app
   - Copy your App ID

3. RUN THE PLATFORM
   - Put all 5 files in the same folder
   - Open index.html in a web browser
   - Click "Connect to Deriv"

4. DEMO VS REAL ACCOUNT
   - Use demo account first (default)
   - Change USE_DEMO to false for real account
   - NEVER share your real account token

5. TROUBLESHOOTING
   - If connection fails: Check your internet
   - If trades fail: Check your balance
   - If errors appear: Open browser console (F12)

6. SECURITY NOTES
   - Never commit config.js with your App ID to GitHub
   - Use environment variables for production
   - Only use your own API tokens
